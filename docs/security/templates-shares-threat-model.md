# 模板与只读分享威胁模型

日期：2026-07-22  
范围：Phase 5 / L5-010、L5-011

| 威胁                                     | 控制                                                                          | 验证                           |
| ---------------------------------------- | ----------------------------------------------------------------------------- | ------------------------------ |
| 私有 Space 经模板泄漏给 workspace        | 私有源只能创建 private 模板；列表/安装按 `created_by` 过滤                    | 同 workspace Viewer 不可见测试 |
| 模板包含成员、Token、Provider Key 或脚本 | 只从白名单结构化规划字段构建；无任意上传/执行；风险元数据显式声明             | 包内容与安装 schema 测试       |
| 安装覆盖用户数据或复用 ID                | 每次生成全新 UUID；走 PlanningService 权限、配额、约束与同事务创建            | 安装 ID 不同及数据库约束测试   |
| 恶意/损坏对象图                          | 安装前重新通过 Pydantic 和领域服务验证；未知字段/类型失败关闭                 | 非法包返回稳定 422             |
| 分享扩大字段范围                         | 服务端只投影七个白名单字段；不包含笔记、附件、成员或引用对象                  | 公开响应字段精确断言           |
| Token 泄漏或数据库被读                   | 256-bit 级 URL-safe Token；仅首次返回；用途域 HMAC 落库；列表/审计不含 Token  | 数据库与审计断言               |
| 撤销后缓存仍可访问                       | 服务端每次校验 status/expiry；`private, no-store`、`no-referrer`；撤销后 404  | 撤销前后公开读取测试           |
| 枚举和流量滥用                           | Token 长度/字符白名单；公开读取按隐私哈希 IP 限流；错误不区分不存在/过期/撤销 | API schema 与限流路径          |
| 跨租户 IDOR                              | 所有管理查询带 workspace；私有模板同时限制创建者；外部 workspace 返回 404     | 跨租户集成测试                 |

公开链接本质上是 bearer secret，获得链接者可在到期或撤销前查看快照；产品不得宣称它等价于成员身份或强访问控制。用户应只选择可公开字段，并通过撤销处理误发。
