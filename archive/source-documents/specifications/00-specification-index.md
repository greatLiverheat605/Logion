# 研途 Lab 规格文档索引

这组文档用于产品、后端、前端、测试和运维协作。发生冲突时，优先顺序如下：

1. 已确认的用户需求和总体规划；
2. PRD 中的业务不变量；
3. 数据字典与同步协议；
4. OpenAPI 契约；
5. 具体实现。

实现不能为了方便而绕过“离线不丢数据、冲突不静默覆盖、AI 只生成草稿、密钥不下发浏览器”等核心原则。

## 统一约定

- 后端：Python 3.12+、FastAPI、SQLAlchemy 2、Alembic、PostgreSQL；
- 客户端：PWA、IndexedDB、Outbox；
- 笔记正文：Yjs/CRDT；
- ID：UUIDv7；
- 时间：服务端 UTC `timestamptz`，客户端本地化；
- 同步协议：`sync-v1`；
- API 前缀：`/api/v1`；
- AI 第一适配器：OpenAI-compatible；
- 日常登录：Passkey；
- 恢复：密码 + TOTP + 一次性恢复码；
- 服务器：云平台个人实例；
- 备份：服务器独立备份卷，并保留手动下载和未来异地扩展；
- Git/Vigils 自动连接器：第一版暂缓但预留接口。
